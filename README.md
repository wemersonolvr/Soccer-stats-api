# Soccer Stats API

Bem-vindo à Soccer Stats API! Esta API foi desenvolvida para fornecer informações detalhadas sobre partidas de futebol, equipes e jogadores. Com ela, você pode acessar dados em tempo real e históricos para análises estatísticas, aplicativos de apostas e jogos virtuais de futebol.

## Descrição

Esta API foi totalmente desenvolvida em TypeScript e oferece uma ampla gama de endpoints para atender às suas necessidades de obtenção, atualização, criação e exclusão de dados relacionados a partidas, times e jogadores.

## Endpoints Disponíveis

- **GET /partidas**: Retorna informações sobre partidas de futebol.
- **GET /times**: Retorna informações sobre equipes.
- **GET /jogadores**: Retorna informações sobre jogadores.

- **PUT /partidas**: Atualiza informações sobre partidas de futebol.
- **PUT /times**: Atualiza informações sobre equipes.
- **PUT /jogadores**: Atualiza informações sobre jogadores.

- **POST /partidas**: Cria novas partidas de futebol.
- **POST /times**: Cria novas equipes.
- **POST /jogadores**: Cria novos jogadores.

- **DELETE /partidas**: Remove partidas de futebol.
- **DELETE /times**: Remove equipes.
- **DELETE /jogadores**: Remove jogadores.
